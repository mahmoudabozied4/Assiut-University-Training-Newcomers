## ğŸš€ Getting Started

### Prerequisites
- C++ compiler (GCC 7.0+ or Clang 5.0+)
- CMake 3.10+ (optional, for building with CMake)
- Basic knowledge of C++ programming

### Compilation

#### Using g++ directly:
```bash
g++ -std=c++17 -O2 "path/to/problem.cpp" -o solution
./solution
```

#### Using CMake:
```bash
mkdir build
cd build
cmake ..
make
```
